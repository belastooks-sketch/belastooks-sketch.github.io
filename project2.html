Project 2: Writing a Program to Encrypt a text file with a Hill Cipher

  • Resume

Description

The assignment was to write a program that encrypts alphabetic letters in a file using the Hill cipher where the Hill matrix can be any size from 2x2 up to 9x9. The program took two command line parameters containing the names of the file storing the encryption key and the file to be encrypted. The program must generate the output to the terminal screen.
